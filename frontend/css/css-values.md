# CSS Values

## Values
```
<integer> 
<number> - DECIMAL   
<dimension> - Number with unit
         <length>
         <angle>
         <time>
         <resolution>

<percentage> - RELATIVE values

Viewport => Available space in browser.


```

## Absolute values:

```
px (PIXELS)
mm,cm,Q,im,pc,pt
s (SECONDS)
deg (GRADES)
```

## Relative values:

```
em - default font size.
rem - default root font size
% - relativo a herencia
vw - ancho del viewport
vh - alto del viewport
```

## Colors:

```

rgb, rgba, hsl, hsla
```
