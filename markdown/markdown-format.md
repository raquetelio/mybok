# Markdown format

https://markdownlivepreview.com/ 



## Scaping characters:

For italic text use backticks (``):
```
_this_is_italic_text_
_`this_is_italic_text`_
```

For tables use backward slash: (\):

```
(x|y) -> NOK
(x\|y) -> OK 
```

| Bracket | Description | 
| ------- | :----------: |
| (x\|y) | Find any of the alternatives specified |
